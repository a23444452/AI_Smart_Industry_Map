# AI 智慧產業地圖

探索台、美、日股產業鏈與投資題材的分析平台。

> ⚠️ **開發中**：本專案目前處於早期建置階段，功能與文件將隨後續 task 陸續補充。

## 預告目錄結構

```
AI_Smart_Industry_Map/
├── backend/        # FastAPI 後端服務
├── frontend/       # React (Vite) 前端
├── data/
│   └── seeds/      # 種子資料
├── docs/           # 設計文件與實作計畫
├── .env.example    # 環境變數範本
└── Makefile        # 常用開發指令
```

## 技術架構

- **前端**：React + Vite
- **後端**：FastAPI
- **資料庫**：SQLite

## 快速開始

1. 複製環境變數範本：

   ```bash
   cp .env.example .env
   ```

2. 後續啟動說明將於相關 task 補充。
